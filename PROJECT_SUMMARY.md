## 🔧 技术栈

- **后端**: Flask 1.1.4 (Python)
- **前端**: Bootstrap 5 + jQuery
- **样式**: 自定义 CSS
- **图标**: Font Awesome
- **通信**: HTTP API + requests

## 🎯 主要特性

1. **安全性**: 管理员登录验证，会话管理
2. **响应式**: 支持各种屏幕尺寸
3. **模块化**: 清晰的代码结构，易于维护
4. **用户友好**: 直观的界面设计，操作简单
5. **功能完整**: 涵盖用户、主题、配置、活动等管理功能

## 🔗 与游戏服务器集成

系统通过 HTTP API 与游戏服务器通信，支持以下接口：

- 用户管理：获取用户列表、用户详情、更新用户信息
- 主题管理：获取主题列表、主题配置
- 配置管理：每日配置、活动配置、AC配置
- 系统管理：系统信息、报告、清理数据

## 📝 使用说明

1. **登录系统**: 使用管理员账户登录
2. **用户管理**: 查看、编辑、管理游戏用户
3. **主题管理**: 配置游戏主题和参数
4. **配置管理**: 调整游戏各种配置
5. **活动管理**: 创建和管理游戏活动
6. **系统监控**: 查看系统状态和统计信息
